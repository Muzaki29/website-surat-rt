# SuratRT — PRODUCT

## Register
product (civic admin portal for RT/RW neighborhood)

## Audience
- **Warga RT 005/RW 002 Kampung Makasar** — ajukan surat, bayar iuran, forum, polling, bantuan
- **Pengurus RT** — verifikasi pengajuan, iuran, monitoring, arsip

## Voice
Jelas, resmi tapi ramah, bahasa Indonesia. Hindari jargon startup ("seamless", "elevate").

## Anti-references
- SaaS gradient hero, purple/blue glow
- Identical 4-column feature cards
- Numbered step circles (01/02/03)
- Uppercase section kickers

## Primary journeys (warga)
1. Daftar → verifikasi RT → login → forum/layanan
2. Ajukan surat → lacak status
3. Bayar iuran → konfirmasi bendahara
4. Bantuan / FAQ / tiket support
